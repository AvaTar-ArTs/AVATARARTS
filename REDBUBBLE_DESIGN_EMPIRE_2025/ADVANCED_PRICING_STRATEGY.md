## üí° PART 3: PRICING PSYCHOLOGY TACTICS

### Tactic #1: The Decoy Effect
**How It Works:** Present 3 price points to make middle one look best

**Example:**
```
Option A: Sticker         $9.95  ‚Üê Budget anchor
Option B: T-Shirt        $19.95  ‚Üê SWEET SPOT (customers choose this)
Option C: Premium Bundle $34.95  ‚Üê Luxury/gift option
```

**Result:** 70% choose Option B (middle price), NOT lowest  
**Revenue:** 0.7 √ó $19.95 = $13.97 avg vs $10.00 if all priced at $9.95

**Implementation:**
- [ ] Create all 3 versions of each design
- [ ] Link to each other: "Customers also bought..."
- [ ] Feature middle tier on homepage

### Tactic #2: The Premium Brand Positioning
**How It Works:** Use language to justify higher price

**Low Pricing Copy:**
> "AI Creator T-Shirt - $12.95"

**Premium Pricing Copy:**
> "AI Creator T-Shirt | Premium Quality | Ethically Sourced | Limited Edition Design - $19.95"

**Result:** Same design, +54% price increase justified by narrative

**Keywords to Use:**
- "Premium"
- "Limited edition"
- "Ethically sourced"
- "Exclusive design"
- "Artist-designed"
- "Quality guaranteed"
- "Collectors' item"

### Tactic #3: The Scarcity Principle
**How It Works:** Create urgency around limited quantity

**Description Copy:**
> "Only 100 of this design will ever be printed. Artist-signed limited edition. Once sold out, never available again."

**Psychology:** FOMO (Fear Of Missing Out) justifies premium price  
**Result:** Expected 20% higher conversion at higher price point

**Implementation:**
- [ ] Add to product description: "Limited to 500 copies"
- [ ] Add to social posts: "Only 47 left!"
- [ ] Update count as you sell

### Tactic #4: The Quality Signaling
**How It Works:** Justify price through quality credentials

**Description Framework:**
```

### Tactic #5: The Bundling Strategy
**How It Works:** Sell multiple designs at discount (but still premium price)

**Example Bundle:**
```

- Image Prompt Generator T-Shirt: $19.95

## üéØ PART 7: PRICE TESTING & OPTIMIZATION

### A/B Testing Framework

#### **Test 1: Price Sensitivity**
**What:** Launch same design at 2 different prices simultaneously
```
Design: "Image Prompt Generator"
Test A: $17.95 (T-shirt)
Test B: $21.95 (T-shirt)

Measure: Sales, revenue per design, profit
Duration: 2 weeks (minimum 30 sales to validate)
```

**Decision Rules:**
- If A wins: Lower price attracts more volume
- If B wins: Market values quality more (premium positioning works)
- If tie: Price doesn't matter, focus on other variables

#### **Test 2: Description Impact**
**What:** Same price, different descriptions
```
Description A: "AI Creator T-Shirt - $19.95"
Description B: "Premium AI Creator Design - Limited Edition, Artist-Designed - $19.95"

Measure: Click-through rate, conversion rate
```

**Decision Rules:**
- If B wins: Use rich, narrative descriptions for all products

#### **Test 3: Format Impact**
**What:** Same design, different product formats
```
Format A: Sticker ($9.95)
Format B: T-shirt ($19.95)
Format C: Hoodie ($24.95)
Format D: Mug ($15.95)

Measure: Sales by format, revenue by format
```

**Decision Rules:**
- Shift inventory to highest-revenue formats
- Reduce/remove low-performing formats

### Implementation:
```
Week 1-4: Test pricing ($17.95 vs $21.95)
Week 5-8: Test descriptions (basic vs narrative)
Week 9-12: Test formats (which sells best)
Week 13+: Implement learnings across all 20+ designs
```
