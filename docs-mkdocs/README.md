# AVATARARTS Documentation - MkDocs Version

This is the MkDocs (Material theme) version of the AVATARARTS documentation.

## Setup

```bash
# Install dependencies
pip install -r requirements.txt

# Serve locally
mkdocs serve

# Build static site
mkdocs build
```

## Features

- Material Design theme
- Dark mode support
- Search functionality
- Responsive design
- Fast builds

## Structure

- `docs/` - Markdown documentation files
- `mkdocs.yml` - Configuration file
- `site/` - Generated static site (after build)
