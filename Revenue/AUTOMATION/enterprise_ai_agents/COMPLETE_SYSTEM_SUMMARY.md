## ðŸš€ Quick Start

### Installation
```bash
cd AUTOMATION/enterprise_ai_agents
pip install -r requirements.txt
```

### Basic Usage
```python
from agent_orchestrator import AgentOrchestrator

orchestrator = AgentOrchestrator()
workflow_id = orchestrator.create_workflow('Enterprise Automation', [
    {
        'id': 'step_1',
        'agent': 'research',
        'task': {'query': 'AI automation trends', 'type': 'trend_analysis'},
        'dependencies': []
    },
    {
        'id': 'step_2',
        'agent': 'content',
        'task': {'topic': 'Enterprise AI', 'format': 'article'},
        'dependencies': ['step_1']
    }
])

result = orchestrator.execute_workflow(workflow_id, parallel=True)
```

### CLI Usage
```bash
# Execute single agent
python main.py agent --type research --task '{"query": "AI trends", "type": "trend_analysis"}'

# Execute workflow
python main.py workflow --file examples/enterprise_automation_workflow.json --parallel

# Use orchestrator
python main.py orchestrate --create "My Workflow" --steps examples/enterprise_automation_workflow.json

# Get statistics
python main.py stats
```
