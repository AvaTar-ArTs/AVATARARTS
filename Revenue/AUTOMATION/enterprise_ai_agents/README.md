### Basic Usage

```python
from enterprise_ai_agents import AgentOrchestrator, ResearchAgent

# Single agent
agent = ResearchAgent('ResearchAgent')
result = agent.run({
    'query': 'AI automation trends',
    'type': 'trend_analysis'
})

# Multi-agent orchestration
orchestrator = AgentOrchestrator()
workflow_id = orchestrator.create_workflow('Enterprise Automation', [
    {
        'id': 'step1',
        'agent': 'research',
        'task': {'query': 'market trends', 'type': 'market_research'},
        'dependencies': []
    },
    {
        'id': 'step2',
        'agent': 'content',
        'task': {'topic': 'Enterprise AI', 'format': 'article'},
        'dependencies': ['step1']
    }
])

result = orchestrator.execute_workflow(workflow_id, parallel=True)
```
