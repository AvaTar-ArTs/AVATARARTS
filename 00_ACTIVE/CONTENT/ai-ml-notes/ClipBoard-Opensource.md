There are several open-source clipboard manager applications for macOS that can save your clipboard history for as long as you want. These applications allow for extensive customization, including how long the clipboard history is saved and various other features. Here are a few notable ones:

### 1. **Maccy**
[Maccy](https://github.com/p0deje/Maccy) is a lightweight clipboard manager for macOS that focuses on simplicity and performance.

- **Features:**
  - Lightweight and fast.
  - Simple and intuitive interface.
  - Searchable clipboard history.
  - Customizable storage duration for clipboard history.
  - Support for ignoring specific apps.
  - Open-source and actively maintained.

#### Installation via Homebrew:
```sh
brew install maccy
```

### 2. **Clipy**
[Clipy](https://github.com/Clipy/Clipy) is another open-source clipboard manager for macOS with a more feature-rich interface.

- **Features:**
  - Unlimited clipboard history.
  - Searchable and categorized clipboard history.
  - Customizable keyboard shortcuts.
  - Paste directly from clipboard history.
  - Supports images and text.
  - Open-source and actively maintained.

#### Installation via Homebrew:
```sh
brew install --cask clipy
```

### 3. **Flycut**
[Flycut](https://github.com/TermiT/Flycut) is a clean clipboard manager for developers, based on Jumpcut.

- **Features:**
  - Simple and easy-to-use.
  - Supports unlimited clipboard history.
  - Keyboard shortcuts for quick access.
  - Paste directly from clipboard history.
  - Open-source.

#### Installation via Homebrew:
```sh
brew install --cask flycut
```

### 4. **Clipboard Manager by Max Mustermann**
[Clipboard Manager](https://github.com/MaxMustermann9/ClipboardManager) is a script-based solution for managing clipboard history on macOS.

- **Features:**
  - Unlimited clipboard history.
  - Text-based history storage.
  - Lightweight and script-based solution.
  - Highly customizable via scripts.

### Conclusion:
Each of these options caters to different needs and preferences. Maccy offers performance and simplicity, Clipy provides a rich feature set, Flycut is great for developers, and Max Mustermannâ€™s script offers lightweight customization. ```sh
