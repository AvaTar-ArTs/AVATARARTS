# Keyboard Shortcuts for Conversation Archive

## System Preferences Setup
1. Open System Preferences > Keyboard > Shortcuts
2. Select "Services" from the left sidebar
3. Find "Open Conversation Archive" under "General"
4. Add your preferred keyboard shortcut (e.g., Cmd+Shift+A)

## Available Shortcuts
- Cmd+Shift+A: Open Archive (recommended)
- Cmd+Space then type "archive": Open via Spotlight
- Cmd+Space then type "Open Archive": Open via Spotlight

## Terminal Aliases (after restarting terminal)
- `archive` or `arch`: Main archive command
- `arch-open`: Open archive
- `arch-update`: Update archive
- `arch-monitor`: Start monitoring
- `arch-status`: Show status
- `arch-search`: Search conversations
