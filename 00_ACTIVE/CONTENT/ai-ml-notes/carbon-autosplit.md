# Carbon Images - Organized Structure

This directory contains all Carbon-generated code images organized by type and purpose.

## Directory Structure

- **original_images/**: Original PNG files generated by Carbon
- **optimized_images/**: PNG files optimized for smaller file sizes
- **webp_images/**: WebP format files for web optimization
- **scripts/**: All automation and optimization scripts
- **reports/**: Analysis reports and documentation
- **analysis/**: Raw analysis data and statistics

## Usage

- **For web use**: Use files from `webp_images/` (smallest file sizes)
- **For print/archive**: Use files from `original_images/` (highest quality)
- **For general use**: Use files from `optimized_images/` (balanced quality/size)

## Scripts Available

- `statistical_carbon_limiter.py`: Intelligent file selection based on size
- `advanced_carbon_optimizer.py`: Comprehensive optimization system
- `intelligent_carbon_splitter.py`: Splits large files into sections
- Various shell scripts for automation and optimization

## Reports

Check the `reports/` directory for detailed analysis and optimization reports.
