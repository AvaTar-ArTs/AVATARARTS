### Monthly Cleanup Script
**Location**: `~/Documents/.config/monthly_cleanup.sh`

```bash
#!/bin/bash
# Automated monthly maintenance

# Delete cache files

find ~/Documents -name "*.pyo" -delete

# Delete system files

# Delete empty directories
```
