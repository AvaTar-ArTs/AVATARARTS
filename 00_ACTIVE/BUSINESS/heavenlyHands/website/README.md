# Heavenly Hands Cleaning — Starter Website

A fast, SEO‑ready static site for local cleaning services (Gainesville/Ocala, FL).

## Features
- Local SEO keywords in title, meta description, headings, and copy
- `schema.org/HouseCleaning` + `FAQPage` JSON‑LD
- Mobile‑first, sticky call‑to‑action bar
- Simple Netlify/Cloudflare/Pages‑ready contact form
- Robots + sitemap
- Accessible tel: and mailto: links

## Quick Start (macOS)
```bash
# Option A: Open locally
open /System/Applications/Safari.app /mnt/data/hhcleaning_site/index.html

# Option B: Simple Python server (http://localhost:8000)
cd /mnt/data/hhcleaning_site
python3 -m http.server 8000

# Option C: Deploy to Netlify
# 1) netlify deploy --dir=/mnt/data/hhcleaning_site
```

## Customize
- Replace `example.com` URLs in `index.html` & `sitemap.xml` with your domain.
- Edit service areas, hours, and copy in `index.html`.
- Add photos and reviews as you collect them to boost E‑E‑A‑T.
