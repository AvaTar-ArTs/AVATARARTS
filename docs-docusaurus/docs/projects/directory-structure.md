# Directory Structure

Detailed breakdown of the AVATARARTS workspace directory organization.

## Numbered Directory System

The workspace uses a numbered system (00-07) for logical organization:

| Directory | Size | Purpose |
|-----------|------|---------|
| **00_ACTIVE** | 1.9 GB | Active projects and development |
| **01_TOOLS** | 13 MB | Tools and utility scripts |
| **02_DOCUMENTATION** | 1.2 MB | Documentation and guides |
| **03_ARCHIVES** | 717 MB | Archived content |
| **04_WEBSITES** | 2.6 GB | Website projects |
| **05_DATA** | 135 MB | Data files and databases |
| **06_SEO_MARKETING** | 36 MB | SEO and marketing systems |
| **07_MISC** | 9.3 MB | Miscellaneous items |

## Navigation Tips

1. **Active Work:** Start in `00_ACTIVE/`
2. **Tools:** Check `01_TOOLS/` for utilities
3. **Documentation:** Browse `02_DOCUMENTATION/`
4. **Old Content:** Look in `03_ARCHIVES/`
5. **Websites:** Explore `04_WEBSITES/`
