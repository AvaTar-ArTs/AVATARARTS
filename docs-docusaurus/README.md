# AVATARARTS Documentation - Docusaurus Version

This is the Docusaurus version of the AVATARARTS documentation.

## Setup

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build

# Serve production build
npm run serve
```

## Features

- React-based documentation
- MDX support (React components in Markdown)
- Built-in search
- Versioning support
- Localization support
- Dark mode
- Responsive design

## Structure

- `docs/` - Markdown documentation files
- `src/` - React components and CSS
- `static/` - Static assets
- `docusaurus.config.js` - Configuration file
- `sidebars.js` - Sidebar navigation configuration
