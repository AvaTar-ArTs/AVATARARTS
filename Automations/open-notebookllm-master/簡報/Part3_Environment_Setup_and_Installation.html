üì• Step 1: Clone the Project

# Clone from GitHub
git clone https://github.com/ChatGPT3a01/open-notebookllm.git

# Enter project directory
cd open-notebookllm
Tip: If Git isn't installed, download the ZIP from GitHub and unzip it.

üîë Step 2: Set Environment Variables

# Windows
copy .env.example backend\.env

# macOS/Linux
cp .env.example backend/.env

Edit backend/.env and add your API keys:

# Choose AI Provider
AI_PROVIDER=gemini

# Gemini settings
GEMINI_API_KEY=your-Gemini-API-Key
GEMINI_MODEL=gemini-2.0-flash

# OpenAI settings (optional)
OPENAI_API_KEY=your-OpenAI-API-Key

üêç Step 3: Install Backend

# Enter backend directory
cd backend

# Create virtual environment
python -m venv venv

# Activate virtual environment
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt
Important: Activate the virtual environment in every new terminal session.

üì¶ Step 4: Install Frontend

Open a new terminal window:

# Enter frontend directory
cd frontend

# Install dependencies
npm install

# Install extra package
npm install -D @tailwindcss/typography
Tip: The frontend uses npm and does not require the Python virtual environment.

üöÄ Step 5: Start Services

Backend (Terminal 1)

cd backend
venv\Scripts\activate
python app.py

Runs at http://localhost:5000

Frontend (Terminal 2)

cd frontend
npm run dev

Runs at http://localhost:3000

Success! Open http://localhost:3000 in your browser to start.

üè† Optional: Ollama Local Models

If you want full offline use, install Ollama:

  1. Download and install from ollama.ai
  2. Start Ollama: ollama serve
  3. Pull a model: ollama pull llama3.2
  4. Set .env: AI_PROVIDER=ollama
Benefits: free, offline, and your data never leaves your machine.
