# Iterative Refinement Prompts

- Make the title more suspenseful, keep under 60 chars.
- Increase contrast and add a single red accent.
- Shift font from shattered to sleek; keep embedded in-world.
- Add a human-scale prop for emotion (flower, letter, locket).
- Push mood from dread → resolve (lighting: warm edge, less blue).
- Shorten description to 240–260 chars, include 3 strongest keywords.
- Generate Option A / Option B titles for A/B testing.
