### When to Pivot
- **CodeCanyon:** If <10 sales/month by week 4 ‚Üí try different 20 scripts
- **Gumroad:** If <5 sales/month by week 4 ‚Üí lower pricing or improve descriptions
- **Email:** If <50 signups/week by week 3 ‚Üí improve lead magnet
- **Product Hunt:** If <200 upvotes by hour 8 ‚Üí may not rank, adjust messaging

### When to Double Down
- **CodeCanyon:** If >30 sales/month ‚Üí upload 20 more immediately
- **Email:** If >100 signups/week ‚Üí double lead magnet promotion
- **Product Hunt:** If in top 5 ‚Üí activate affiliate network
- **Affiliate:** If 10+ active affiliates ‚Üí increase commission

### When to Kill
- **Any initiative:** If 4+ weeks with zero traction ‚Üí kill and pivot
- **Paid ads:** If CPA > LTV by 2x after 50+ conversions ‚Üí pause
- **DFY service:** If <1 client/month after 8 weeks ‚Üí pivot or kill
