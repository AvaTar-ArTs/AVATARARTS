## Implementation Workflows

### New Client Workflow
1. **Discovery** (Week 1)
   - Use intake form from Business Templates
   - Initial audit with Analysis Template
   - Competitive research
   
2. **Strategy** (Week 2)
   - Technical audit with Technical Strategy
   - Content audit with Content Strategy
   - Create proposal from Business Templates

3. **Implementation** (Week 3-8)
   - Fix technical issues
   - Optimize existing content
   - Create new content
   - Build backlinks

4. **Optimization** (Ongoing)
   - Monthly reporting
   - Keyword tracking
   - Content updates
   - Performance monitoring

### Content Creation Workflow
1. **Research** - Use keyword tools from Resources
2. **Planning** - Use content brief template
3. **Writing** - Follow SEO writing checklist
4. **Optimization** - Implement on-page SEO
5. **Distribution** - Multi-channel promotion
6. **Analysis** - Track performance metrics
