## Keyword Research Workflow

### Step 1: Brainstorm Seed Keywords

**Method:**
1. List your products/services
2. List customer pain points
3. List common questions you receive
4. Check competitor websites
5. Use Google autocomplete

**Template:**
```
Product/Service: ____________
- Seed keyword 1: ____________
- Seed keyword 2: ____________
- Seed keyword 3: ____________

Customer problem: ____________
- Related search: ____________
- Related search: ____________
```

### Step 2: Expand Keyword List

**Tools to Use:**
- **Free:** Google Keyword Planner, Ubersuggest, AnswerThePublic
- **Paid:** Ahrefs, SEMrush, Moz Keyword Explorer

**Data to Collect:**
- Search volume (monthly)
- Keyword difficulty (0-100)
- Cost per click (CPC)
- Search intent (informational, navigational, commercial, transactional)
- SERP features (featured snippets, People Also Ask, etc.)

**Keyword Research Spreadsheet:**
```
| Keyword | Volume | Difficulty | CPC | Intent | Current Rank | Priority |
|---------|--------|------------|-----|--------|--------------|----------|
| seo services | 12,000 | 75 | $15 | Commercial | Not ranking | High |
| how to do seo | 8,000 | 45 | $8 | Informational | #42 | High |
| local seo tips | 2,400 | 30 | $6 | Informational | Not ranking | Medium |
```

### Step 3: Analyze Search Intent

**Intent Types:**

**Informational:**
- User wants to learn something
- Keywords: "how to," "what is," "guide," "tutorial"
- Content: Blog posts, guides, tutorials
- Example: "how to do keyword research"

**Navigational:**
- User wants to find a specific website
- Keywords: brand names, specific product names
- Content: Homepage, landing pages
- Example: "semrush login"

**Commercial Investigation:**
- User is researching before buying
- Keywords: "best," "vs," "review," "comparison"
- Content: Comparison pages, reviews, roundups
- Example: "best seo tools 2025"

**Transactional:**
- User is ready to buy/take action
- Keywords: "buy," "discount," "coupon," "order"
- Content: Product pages, service pages
- Example: "buy semrush subscription"

### Step 4: Competitor Analysis

**What to Analyze:**
1. Keywords they rank for (that you don't)
2. Content types they create
3. Content length and depth
4. Backlinks to their top content
5. Social shares and engagement

**Competitor Analysis Template:**
```
Competitor: ____________

Top Ranking Keywords:
1. ____________ (Position #___, Volume: ___)
2. ____________ (Position #___, Volume: ___)
3. ____________ (Position #___, Volume: ___)

Content Strategy:
- Average word count: ___
- Update frequency: ___
- Content types: ____________
- Unique angles: ____________

Opportunities:
- Keywords they don't target: ____________
- Content gaps: ____________
- Weaknesses to exploit: ____________
```

### Step 5: Prioritize Keywords

**Priority Matrix:**

**High Priority:**
- High search volume (>1,000/month)
- Low-medium difficulty (<50)
- High relevance to business
- Strong commercial intent

**Medium Priority:**
- Medium search volume (100-1,000/month)
- Medium difficulty (50-70)
- Good relevance
- Mixed intent

**Low Priority:**
- Low search volume (<100/month)
- High difficulty (>70)
- Or: Low relevance
- Or: Unclear intent

### Writing Guidelines

**Tone:** Professional but conversational  
**Reading Level:** 8th-10th grade (Flesch Reading Ease: 60-70)  
**Perspective:** Second person ("you") or first person plural ("we")

**Style Notes:**
- Short paragraphs (2-4 sentences)
- Bullet points for lists
- Bold key terms
- Include examples
- Use transition words

**Things to Avoid:**
- Passive voice (keep under 10%)
- Jargon (unless defined)
- Fluff and filler content
- Keyword stuffing
- Clickbait

- [ ] Keyword research completed

**Content Structure:**
- [ ] Compelling H1 with primary keyword

## Content Optimization

### Optimizing Existing Content

**When to Optimize:**
- Content is 12+ months old
- Rankings have dropped
- Information is outdated
- Competitors have better content
- Low engagement metrics

**Optimization Checklist:**

**1. Content Refresh:**
- [ ] Update statistics and data
- [ ] Replace outdated screenshots
- [ ] Add recent examples
- [ ] Update "last updated" date
- [ ] Add new sections if needed

**2. SEO Improvements:**
- [ ] Improve title tag (CTR optimization)
- [ ] Rewrite meta description
- [ ] Add missing headers
- [ ] Optimize images
- [ ] Add internal links
- [ ] Update keyword targeting

**3. User Experience:**
- [ ] Improve readability
- [ ] Add table of contents
- [ ] Include FAQ section
- [ ] Add more visuals
- [ ] Improve formatting

**4. Engagement:**
- [ ] Add compelling CTA
- [ ] Include downloadable resources
- [ ] Add comment section
- [ ] Include social sharing buttons
- [ ] Add related content recommendations

### Content Pruning

**When to Prune:**
- Content gets <10 views/month
- No rankings for target keywords
- Outdated and not worth updating
- Duplicate or thin content
- Off-brand or low-quality

**Pruning Options:**
- **Update:** Refresh and republish
- **Merge:** Combine with similar content
- **Redirect:** 301 redirect to related page
- **Noindex:** Keep page but remove from index
- **Delete:** Remove completely (last resort)

## Content Types & Best Practices

### Blog Posts

**Optimal Length:** 1,500-2,500 words (varies by topic)

**Structure:**
1. Compelling headline
2. Strong introduction (hook + problem + solution)
3. Well-organized body (H2/H3 hierarchy)
4. Actionable takeaways
5. Strong conclusion with CTA

**Best For:**
- How-to guides
- List posts ("Top 10...")
- Tutorials
- Industry insights

### Long-Form Guides

**Optimal Length:** 3,000-5,000+ words

**Structure:**
1. Comprehensive table of contents
2. Executive summary
3. Detailed sections with examples
4. Data and statistics
5. Case studies
6. Downloadable PDF version

**Best For:**
- Ultimate guides
- Complete tutorials
- Industry reports
- Educational content

### Case Studies

**Optimal Length:** 1,000-2,000 words

**Structure:**
1. Client background
2. Challenge/problem
3. Solution/approach
4. Implementation
5. Results (with data)
6. Key takeaways

**Best For:**
- Demonstrating expertise
- Building trust
- Converting prospects
- Social proof

### Infographics

**Optimal Size:** 800-1000px wide, vertical

**Elements:**
- Clear title
- Data visualization
- Minimal text
- Source citations
- Branded design
- Embed code for sharing

**Best For:**
- Complex data
- Statistics
- Process flows
- Comparisons

### Videos

**Optimal Length:** 
- Tutorials: 5-10 minutes
- Explanations: 2-5 minutes
- Product demos: 3-5 minutes

**Requirements:**
- Script or outline
- High-quality visuals
- Clear audio
- Subtitles/captions
- Compelling thumbnail
- Optimized title/description

## Content Distribution Strategy

### Owned Channels

**Website/Blog:**
- Publish all content here first
- Optimize for SEO
- Add internal links
- Include social sharing

**Email Newsletter:**
- Weekly digest of new content
- Segmented by interest
- Personalized recommendations
- Clear CTAs

**Social Media:**
- Share across all platforms
- Platform-specific formats
- Multiple times per piece
- Engage with comments

### Earned Channels

**Guest Posting:**
- Target high-authority sites
- Include author bio with link
- Promote on social media
- Build relationships

**PR & Media:**
- Pitch to journalists
- Respond to HARO requests
- Create newsworthy content
- Build media relationships

**Influencer Outreach:**
- Identify relevant influencers
- Build relationships
- Provide value first
- Co-create content

### Paid Channels

**Social Media Ads:**
- Promote top content
- Target specific audiences
- A/B test ad creative
- Track ROI

**Content Discovery:**
- Outbrain, Taboola
- Native advertising
- Retargeting
