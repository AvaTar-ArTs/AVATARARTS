## What is Hugo?

Hugo is a fast and modern static site generator written in Go. It's designed to be:

- **Fast**: Builds sites in milliseconds
- **Flexible**: Works with any content
- **Easy**: Simple to use and deploy
- **Powerful**: Rich templating and shortcodes

## Getting Started

To create a new post, simply run:

```bash
hugo new posts/my-new-post.md
```

Then edit the file in `content/posts/` and add your content.

This blog includes:
