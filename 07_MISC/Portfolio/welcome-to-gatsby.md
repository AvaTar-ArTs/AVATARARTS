## What is Gatsby?

Gatsby is a React-based static site generator that creates fast, modern websites. It's perfect for blogs, portfolios, and business websites.

### Key Features

- **Blazing Fast**: Optimized for performance
- **Modern Stack**: Built with React and GraphQL
- **Rich Ecosystem**: Thousands of plugins
- **SEO Friendly**: Built-in optimization
- **Developer Experience**: Hot reloading and great tooling

## Getting Started

To create a new post, simply add a new markdown file to the `content/blog` directory with the following frontmatter:

```markdown
---
title: "Your Post Title"
date: "2025-01-16"
description: "A brief description"
tags: ["tag1", "tag2"]
---

Your content here...
```
